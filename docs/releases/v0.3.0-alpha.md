# Hermes Toolkit v0.3.0-alpha

Hermes Toolkit v0.3.0-alpha is the first complete alpha dashboard release of the project.

This release turns Hermes Toolkit from an early settings helper into a broader local operations console for Hermes-style LLM environments. The focus of this alpha is safe visibility, configuration awareness, diagnostics, and read-only operational insight.

## Safety model

Hermes Toolkit v0.3.0-alpha is intentionally conservative.

Most pages are read-only dashboards. They summarize configuration, logs, runtime state, and inferred activity without modifying local services or user data.

The Models page supports safe config-only model switching. It does not install models, remove models, restart providers, or modify external services.

The Doctor page can run diagnostics, but the diagnostic action is non-destructive.

The Cost page provides estimates only. It is not billing data and should not be treated as an invoice, statement, or provider-side usage record.

The Skills and Memory pages are visibility foundations only. They do not install, remove, export, inspect deeply, or mutate plugin or memory stores.

## Known limitations

This release is still an alpha foundation.

## Upgrade notes

This release does not require a migration step.

Users should still treat Hermes Toolkit as an alpha tool and review configuration changes carefully before relying on it in production-like environments.
